# Estratégia de Marketing para Desenvolvedores

## Posicionamento
"Framework React modular, open source e pronto para produção em Node + Edge".

## Canais prioritários

1. GitHub (README, releases, discussions).
2. X/Twitter e LinkedIn com conteúdo técnico curto.
3. YouTube/live-coding para lançamentos e deep-dives.
4. Dev.to/Hashnode com tutoriais de migração.

## Plano de conteúdo

- Série "Building in public" (semanal).
- Benchmarks transparentes (mensal).
- Estudos de caso de apps reais (bimestral).

## Growth loops

- Cada release gera: changelog + post técnico + demo curta.
- Cada exemplo novo vira tutorial e template reutilizável.
- Cada issue resolvida relevante vira "learning snippet".

## Atração de empresas

- Página de adoção com SLA e política de versões.
- Showcase de empresas usuárias.
- Guia de segurança e compliance.
