# Contribuindo para o Nextify.js

Obrigado por considerar contribuir com o Nextify.js! Nosso objetivo é construir um framework React open source de alto impacto com colaboração saudável e previsível.

## Setup local

```bash
npm install
npm run build
```

## Fluxo de contribuição

1. Abra/associe uma issue.
2. Crie branch: `feat/<descricao>` ou `fix/<descricao>`.
3. Faça commits pequenos e semânticos.
4. Rode checks locais.
5. Abra PR com contexto, trade-offs e evidências.

## Convenções

- Commits no padrão Conventional Commits (`feat:`, `fix:`, `docs:`...).
- Mudanças em `core` devem incluir teste de integração quando aplicável.
- Mudanças breaking devem atualizar docs e changelog.

## Labels

- `good first issue`: tarefas de entrada.
- `help wanted`: tarefas prontas para colaboração externa.
- `priority:high`: bugs críticos.
- `rfc`: discussão de design antes da implementação.

## SLA de manutenção

- Primeira resposta em issue/PR: até 48h.
- Revisão técnica inicial: até 5 dias úteis.

## Código de conduta

Ao contribuir, você concorda com o [`CODE_OF_CONDUCT.md`](./CODE_OF_CONDUCT.md).
