# Exemplos oficiais do Nextify.js

- `blog/`: blog com rotas dinâmicas e API route.
- `saas-dashboard/`: área autenticada de produto SaaS.
- `ecommerce-storefront/`: catálogo e páginas de produto.
- `reference-app/`: fluxo ponta-a-ponta (SSR streaming, API, middleware e cache tags) para smoke E2E.

Cada exemplo deve priorizar legibilidade para onboarding de novos contribuidores.
